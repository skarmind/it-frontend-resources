# Описание

### Что представляет из себя проект

Это репозиторий дизайн-системы, основанный на макете Figma (см. ниже). Здесь собраны переменные дизайн системы в форматах CSS, SCSS и JS, а так же вспомогательные элементы дизайн-системы, на основе которых разрабатываются дизайн-проекты различных веб-приложений внутри компании Playrix. Подробнее об этих элементах можно узнать в соответствующем [разделе](elementy-dizain-sistemy/) документации.

### Какую проблему этот проект решает

В компании Playrix существует несколько команд разработки и большое количество проектов со своим дизайном, а так же различными технологическими стеками Frontend части (Vue 2, Vue 3, Vuetify, React). При разработке новых проектов, а так же при изменении дизайна старых, приходится производить изменения в каждом проекте отдельно. &#x20;
