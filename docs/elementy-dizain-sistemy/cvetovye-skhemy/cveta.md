# Цвета

#### Базовая палитра

| CSS          | JS    | Назначение                                                              |
| ------------ | ----- | ----------------------------------------------------------------------- |
| --plx-base-1 | base1 | Основной фон страницы и заполнение инвертированных иконок               |
| --plx-base-2 | base2 | Вспомогательный цвет фона страницы, плитки основного фона и фоны иконок |
| --plx-base-3 | base3 | Hover фоны для плиток                                                   |
| --plx-base-4 | base4 | Границы таблиц, отдельные участки и блоки                               |
| --plx-base-5 | base5 | Hover границы участков и блоков                                         |
| --plx-base-6 | base6 | Hover интерактивных интерфейсных иконок                                 |
| --plx-base-7 | base7 | Блоки с инвертированным фоном                                           |
| --plx-base-8 | base8 | Альтернативный цвет интерфесных иконок                                  |
| --plx-base-9 | base9 | Иконки поверх инвертированного фона                                     |

#### Поверхности

| CSS             | JS       | Назначение                                           |
| --------------- | -------- | ---------------------------------------------------- |
| --plx-surface-1 | surface1 | Основной фон страницы                                |
| --plx-surface-2 | surface2 | Вспомогательный фон, участки поверхн основного фона  |
| --plx-surface-3 | surface3 | Третичный фон, участки поверхн вспомогательного фона |
| --plx-surface-4 | surface4 | Цвет фона шапки таблицы                              |
