# Отступы

{% hint style="info" %}
В интерфейсах используются отступы кратные 4, за исключением 3xs размера. Это размеры отступов внутри блоков (padding) и между блоками (margin)
{% endhint %}

| -------------- | --------- | -------- |
| --plx-space-xs | spaceXs   | 2px      |
| --plx-space-1  | space1    | 4px      |
| --plx-space-2  | space2    | 8px      |
| --plx-space-3  | space3    | 12px     |
| --plx-space-4  | space4    | 16px     |
| --plx-space-5  | space5    | 20px     |

| --plx-space-16 | space16   | 64px     |
| --plx-space-20 | space20   | 80px     |
| --plx-space-30 | space30   | 120px    |
