# Продолжительности

{% hint style="info" %}
Продолжительности определяют скорости анимации элементов дизайн-системы
{% endhint %}

| CSS версия | JS версия | Скорость |
| ---------- | --------- | -------- |
| --fastest  | fastest   | 0.1s     |
| --faster   | faster    | 0.2s     |
| --fast     | fast      | 0.4s     |
| --slow     | slow      | 0.6s     |
| --slower   | slower    | 0.8s     |
| --slowest  | slowest   | 1s       |
