# Продолжительности

{% hint style="info" %}
Продолжительности определяют скорости анимации элементов дизайн-системы
{% endhint %}

| CSS           | JS      | Скорость |
| ------------- | ------- | -------- |
| --plx-fastest | fastest | 0.1s     |
| --plx-faster  | faster  | 0.2s     |
| --plx-fast    | fast    | 0.4s     |
| --plx-slow    | slow    | 0.6s     |
| --plx-slower  | slower  | 0.8s     |
| --plx-slowest | slowest | 1s       |
