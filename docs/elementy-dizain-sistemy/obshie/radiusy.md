# Радиусы

{% hint style="info" %}
Закругления используются для того, чтобы облегчить восприятие информации и отличить небольшие управляющие элементы  от крупных объектов. Типичными примерами элементов,  которым нужны скругления, являются кнопки или флаги. Величина скруглений кратна 4.
{% endhint %}

| CSS версия        | JS версия  | Значение |
| ----------------- | ---------- | -------- |
| --plx-radius-none | radiusNone | 0px      |
| --plx-radius-xs   | radiusXs   | 2px      |
| --plx-radius-s    | radiusS    | 4px      |
| --plx-radius-m    | radiusM    | 8px      |
| --plx-radius-l    | radiusL    | 12px     |
| --plx-radius-xl   | radiusXl   | 24px     |
| --plx-radius-full | radiusFull | 50%      |
