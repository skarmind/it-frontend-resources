# Разделители

{% hint style="info" %}
Разделительные линии используются для группировки содержимого в таких компонентах, как списки, карточки, таблицы.
{% endhint %}

| CSS             | JS       | Толщина |
| --------------- | -------- | ------- |
| --plx-divider-s | dividerS | 1px     |
| --plx-divider-m | dividerM | 2px     |
| --plx-divider-l | dividerL | 4px     |
