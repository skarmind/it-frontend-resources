Введение →

CSS переменные

Имеется возможность переключать темы через изменение атрибута color-scheme в соответствующее теме значение.

JS переменные

Подключаются в проект через JavaScript import, что открывает дополнительные возможности по изменению состояний приложения и взаимодействию с библиотеками и фреймворками.

SCSS переменные

Компилируются раз и навсегда при старте проекта. Выгодно использовать, когда не нужно изменять значение переменных в процессе работы приложения.
