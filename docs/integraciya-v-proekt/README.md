# Как изменять данные

Источником данных для генерации файлов являются JS файлы модулей, хранящиеся в папке src. Подробнее можно узнать [здесь](edinyi-istochnik-dannykh.md).

На данный момент процесс генерации переменных дизайн-системы можно разбить на два этапа:

1. [Создание CSS переменных в папке src/styles](./#etap-1)
2. [Компиляция готовой сборки дизайн-системы для импорта в папке dist](etap-2-kompilyaciya-i-perenos.md)
