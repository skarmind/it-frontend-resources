# ЭТАП 2: Компиляция и перенос

Второй этап состоит из нескольких частей:

1. Сборка JavaScript исходников с помощью npm пакета microbundle в готовые для импорта в файлы. Предполагается возможность импорта 3 способами: ES модуль (props.module.js), CommonJS модуль (props.cjs) и UMD модуль (props.umd.js). Перед обработкой microbundle обрабатываются скриптом index.js, который изменяет их имена на camelCase версию.
2. &#x20;Постобработка CSS файлов с помощью POSTCSS. Этот процесс предполагает минификацию и перенос всех CSS переменных и кода подключения шрифтов в виде файла props.min.css в папке dist/style
3. Перенос папки со шрифтами и scss переменными и миксинами в папку dist.

Запуск данных операций происходит с помощью команды `npm run bundle`
