# Разделение по темам

Токены, поступающие из плагина Figma Tokens разделены по темам. Причастность токена к определенной теме влияет на его использование в дизайне проектов. Использование тем в системе токенов зависит от способа их подключения в проект. Более подробно темы описаны в отдельных способах подключения

# Разделение по типам

В процессе генерации переменных токены, экспортируемые из плагина Figma Tokens, приводятся к соответствующему типу. Каждый тип берет свое название из названия токена в стиле PascalCase (например токен Border radius преобразуется в токен с типом "BorderRadius"). Так же для удобства использования токены с градиентами преобразуют свой тип из 'color' в "gradient". Токены и их типы можно увидеть в таблице:

Токен Figma Тип Значения
Sizing Sizing Ширины или высоты
Spacing Spacing Отступы
Color Color Цвета заливки или границы
Color Gradient Градиенты
Border radius BorderRadius Радиусы скруглений
Border width BorderWidth Ширины границ
Opacity Opacity Прозрачности
Box shadow BoxShadow Комбинация значений тени контейнеров
Typography Typography Комбинация значений типографики

# Подключение переменных

Для интеграции в проект itwa-design-tokens имеет 3 возможных способа:

  1. JavaScript переменные
  2. CSS переменные
  3. SCSS переменные

Для удобства будем назвать каждый способ подключению платформой. Для каждой платформы есть 2 пути подключения:

  1. Подключение всех токенов через файл common.{extension}, лежащий в папке соответствующей платформы. В этом файле собраны все переменные, не зависимые от цветовой темы, а также переменные светлой темы. Рядом в той же папке есть файлы с переменными отдельных тем, таких как dark.{extension}. Они подключаются отдельно, так как нужны не во всех проектах.
  2. Подключения отдельных типов токенов, лежащих в папке parts. Это сделано для того, чтобы можно было импортировать не все токены целиком, а только отдельные типы. Файлы для импорта именованы в kebab-case формате согласно шаблону {theme}-{type}.{extension}. Например цвета темной темы JavaScript переменных сохранены в папке parts под именем dark-colors.js, токены теней светлой темы CSS переменных в файле light-box-shadow.js.

# Подключение JavaScript переменных

# Пример подключения всех токенов основной светлой темы и токенов темной темы:

import * as common from 'itwa-design-tokens/dist/js/common';
import * as dark from 'itwa-design-tokens/dist/js/dark';

# Пример подключения отдельных типов токенов:

import * as spaces from 'itwa-design-tokens/dist/js/parts/global-spaces';
import * as lightColors from 'itwa-design-tokens/dist/js/parts/light-colors';

# Подключение CSS переменных

# Пример подключения всех токенов основной светлой темы и токенов темной темы:

@import "itwa-design-tokens/dist/css/common.css";
@import "itwa-design-tokens/dist/css/dark.css";

# Пример подключения отдельных типов токенов:

@import 'itwa-design-tokens/dist/css/parts/global-opacity.css';
@import 'itwa-design-tokens/dist/css/parts/dark-colors';

Внимание

Для переключения цветовых схем необходимо установить атрибуту color-scheme корневого элемента html значение имени темы. Это автоматически изменит значения CSS переменных соответственно их цветовой схеме.

<html lang="ru" color-scheme="dark">
