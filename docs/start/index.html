# Описание

# Что такое токены дизайна?

Токены дизайна — это набор базовых переменных визуального языка — отступы, цвета, типографика, тени, градиенты и т. д. — представленный в виде данных. Мы используем их вместо жестко закодированных значений (например, шестнадцатеричных значений для цвета или значений пикселей для отступов), чтобы поддерживать масштабируемую и согласованную визуальную систему для разработки пользовательского интерфейса.

# Что представляет из себя проект

Это репозиторий дизайн-системы, основанный на макете Figma (opens new window). Здесь собраны переменные дизайн системы в форматах CSS, SCSS и JS, на основе которых разрабатываются дизайн-проекты различных веб-приложений внутри компании Playrix.

Источником данных для генерации переменных является JSON объект, поступающий из программы Figma с помощью плагина Figma Tokens.

Подробнее об этих элементах можно узнать в соответствующем разделе документации.

# Какую проблему этот проект решает

В компании Playrix существует несколько команд разработки и большое количество проектов со своим дизайном, а так же различными технологическими стеками Frontend части (Vue 2, Vue 3, Vuetify, React). При разработке новых проектов, а так же при изменении дизайна старых, приходится производить изменения в каждом проекте отдельно. При применении элементов дизайн системы в Frontend частях проектов лучшей практикой является формирование единого источника данных.

Проект itwa-design-tokens позволяет получить на выходе переменные дизайн-системы в нескольких форматах, удобных для интеграции в различные технологические стеки.
